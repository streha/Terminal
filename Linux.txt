Команды терминала Linux

1. pwd - Посмотреть, где я
2. mkdir - Создать папку
3. cd - Зайти в папку
4. mkdir foldername1 foldername2 foldername3 - создать три папки
5. cd foldername1 - зайти в папку
6. touch Filename1.txt Filename2.txt Filename3.txt Filename4.json Filename5.json - создать 3 файла в формате txt и 2 файла в формате json
7. mkdir foldername1 foldername2 foldername3 - создать три папки

8. ls -la - вывести содержимое папки
   ls - список файлов в каталоге
   ls -l - список файлов в каталоге в табличном виде
   ls –a - список файлов в каталоге, включая скрытые папки
   ls -author - список файлов по имени создателя
   ls -cl - список файлов с сортировкой по имени
   ls -ct - список файлов с сортировкой по времени создания (сначала самые новые)
   ls -s - список файлов с размером файла
   ls * - список текстовых файлов

9. vim Filename1.txt - открыть файл txt
10. Для того, чтобы написать что-то в vim-редакторе, при открытии надо нажать клавишу I (Insert)
11. :w - cохранить в vim-редакторе, :q - выйти из vim-редактора
12. c.. - перемещение из папки на уровень выше

13. mv Filename1.txt Filename2.txt ./foldername1 - переместить 2 файла в другую папку в этой же директории
    mv Filename 1.txt Filename.txt ../foldername1 - переместить 2 файла в папку в директорию на уровень выше

14. cp Filename1.txt Filename2.txt ./folfername2 - скопировать 2 файла в папку в этой же директории

15. find -name filename1 - найти файл по имени (работает, если запустить Git Bash от имени администратора)
    find -name "*.txt" - найти файл, заканчивающихся на .txt

16. tail -f или sudo tail -f - команда для отслеживания лог-файлов в реальном времени
    grep -

17. cat filename - вывести весь текстовый файл в терминал
    head -n filename - вывести несколько первых строк из текстового файла

18. tail -n filename - вывести несколько последних строк из текстового файла

19. less filename - посмотреть содержимое длинного файла
20. date - вывести дату и время
